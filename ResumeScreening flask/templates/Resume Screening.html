What we've done for people

A service which can be used by Talent acquisition team to filter resumes based on job description before passing them to technical team for further processing. Basic concept here is, when we upload a job description and a bunch of resumes to the tool, it should rank resumes in descending order according to the percentage it matches with the job description.
